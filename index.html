Flux - Static Site Generator

Flux is an opinionated Static Site Generator written in Go programming language. It is a command line utility.

Features (v1)

  • Support for front matter
  • Syntax highlighting
  • Support for Jinja templates
  • Support for Markdown files
  • Archive lists
  • Live reloading with http server
  • Support for Pug
  • Support for CSS language extensions
